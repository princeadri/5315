What is Docker

Docker is a container virtualization technology that enable developers to create, deploy, and run applications. It can simultaneously run different containers and allows applications to use the same kernel as the system they are running on. What is VM (virtual machine)

A virtual machine is a computer software that behaves like an actual, physical computer. It allows the user to run other operating software (OS) within the current OS in use. It is possible to simultaneously run multiple virtual machines within a computer, but the number of virtual machines one can install is limited to the storage available for them. Reference: https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/ and https://www.makeuseof.com/tag/virtual-machine-makeuseof-explains/
